**开发 OpenClaw Telebot Skill**（定时任务）
    - [ ]  每日自动抓取加密新闻（来源：CoinDesk、CoinTelegraph、Decrypt、The Block）
    - [ ]  用 AI 生成**小白友好型摘要**：
        - 📰 **大白话总结**：用一句话说清楚这条新闻在讲什么（不用术语）
        - 📖 **术语高亮**：标注新闻中的关键术语，附上一行解释（如："Layer 2 = 帮主链分担压力的扩容网络"）
        - 🏷️ **分类标签**：给新闻打标签便于积累认知（`#监管` `#比特币` `#以太坊` `#DeFi` `#新项目` `#宏观经济`）
        - 💡 **延伸一问**：每条新闻附带一个思考题，引导主动思考（如："为什么 ETF 获批会影响比特币价格？"）
    - [ ]  Telebot 每天早上定时推送摘要
    - [ ]  推送分为两个区块：
        - 🟢 **必读**（2-3 条最重要的，小白也能看懂）
        - 🔵 **进阶**（需要一定基础才能理解，作为挑战阅读）
